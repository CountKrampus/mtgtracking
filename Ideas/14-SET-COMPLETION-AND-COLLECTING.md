# Set Completion & Collecting Ideas

## High Priority

### Set Tracker Improvements
- **Detailed Set View**: Click a set to see full card checklist with owned/missing indicators
- **Missing Cards List**: For each set, show exactly which cards you're missing with current prices
- **Set Completion Cost**: Calculate how much it would cost to complete each set at current prices
- **Cheapest Missing Cards**: Sort missing cards by price to find easy pickups
- **Set Rarity Breakdown**: Show completion by rarity within each set (all commons? missing 3 rares?)
- **Set Filter by Type**: Filter set tracker to show only Standard-legal sets, Commander sets, Masters sets, etc.
- **Set Search**: Search for a specific set in the tracker
- **Set Sorting Options**: Sort sets by completion %, name, release date, or number of owned cards

### Collection Goals
- **Goal System**: Create custom collection goals with progress tracking
  - "Complete all Ravnica sets"
  - "Own every Lightning Bolt printing"
  - "Collect all Planeswalkers"
  - "Own a card from every set"
- **Goal Templates**: Pre-built goals for common collecting objectives
- **Goal Progress Notifications**: Alert when you're close to completing a goal
- **Goal Sharing**: Share collection goals with friends

### Print Variants & Special Editions
- **Track Print Variants**: Distinguish between regular, foil, etched foil, extended art, borderless, showcase, retro frame
- **Variant Checklist**: For collectors who want every variant of a card
- **Secret Lair Tracking**: Track Secret Lair drops separately with their unique art
- **Promo Tracking**: Track promo cards (FNM promos, buy-a-box, prerelease, etc.)
- **Masterpiece Tracking**: Track Masterpiece/Expedition/Invention special versions
- **Serial Numbered Cards**: Track serialized cards with their specific numbers

## Medium Priority

### Collecting Features
- **Card Cycle Tracker**: Track completion of MTG cycles (Swords cycle, Titan cycle, Command cycle, etc.)
- **Planeswalker Collection**: Dedicated view for tracking all Planeswalkers owned
- **Artist Collection**: Track how many unique card arts you own per artist
- **Reserved List Tracker**: Highlight and track Reserved List cards in your collection
- **Vintage/ABG Tracker**: Special tracking for Alpha, Beta, Unlimited, and other early sets
- **Full Art Land Collection**: Track your full-art basic land collection
- **Mana Curve by Set**: See the mana curve of your owned cards per set

### Comparison & Discovery
- **Set Value Comparison**: Compare the total value of cards you own across different sets
- **Most Complete Sets**: Rank your sets by completion percentage
- **Set Release Timeline**: Visual timeline of all sets with your collection overlaid
- **New Set Alerts**: Notification when a new set is released with link to start tracking
- **Set EV Calculator**: Calculate expected value of opening boosters based on current prices
- **Box Opening Tracker**: Log booster box openings with pulls and calculate actual value received

### Visual Collecting
- **Binder Page View**: View your collection organized as visual binder pages (3x3 grid per page)
- **Set Poster**: Generate a poster-style image of all owned cards from a set arranged in collector number order
- **Showcase Wall**: Display your most valuable or favorite cards in a visual showcase
- **3D Binder**: Interactive 3D binder flip-through experience

## Lower Priority

- **WUBRG Completion**: Track how many unique cards you own per color across all sets
- **Card Frame Tracking**: Track old border vs modern border vs retro frame cards
- **Misprint Tracking**: Log and track misprinted cards
- **Signed Card Tracking**: Track artist-signed cards with authentication details
- **Card Grading Queue**: Queue cards for grading submission; track grading status
- **Collecting Achievements**: Unlock achievements for collecting milestones (complete a rare set, own 500 creatures, etc.)
